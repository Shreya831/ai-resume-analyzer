# 🚀 AI Resume Analyzer

An AI-powered tool that compares your resume with any job description and shows exactly what’s missing.

## 🔍 Features
- Resume vs Job Matching
- Semantic AI scoring
- Missing skills detection
- Works for any domain

## 🧠 Tech Stack
- Sentence Transformers
- KeyBERT
- Gradio
- Python

## 🚀 Demo
Upload your resume and paste a job description to get instant insights.

## 💡 Example Output
- Match Score: 32%
- Missing Skills: Docker, Kubernetes, APIs

```bash
pip install -r requirements.txt
